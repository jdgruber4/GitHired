CPSC 314 Final Project Presentation

GitHired

JD Gruber

GitHired is a streamlined job surfing application. GitHired brings the jobs to you with an easy to use interface to help you get hired.

Functional Requirements:

  1. Users must be able to view the title of work they are looking for.

  2. Users must be able to seelocation of Job posting

  3. Users must be able to seethe day of job posting

  4. Users must be able to surf multiple pages of jobs

  5. Users must be able to see the company that posted the job

  6. The application will automatically update to refresh job postings.

  7. The application will store every job uniquely in the database in separatediv components.

  8. Users must be able to retrieve cached job postings from previous search/scrape results.

  9. Job objects must not be maintained for longer than they are available on the given job site.

  10. The application will utilize job hosting sites APIto manage and present all applicable data to the user.
